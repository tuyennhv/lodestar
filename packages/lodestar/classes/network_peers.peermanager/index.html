    Class: PeerManager¶

    Performs all peer managment functionality in a single grouped class: - Ping peers every PING_INTERVAL_MS - Status peers every STATUS_INTERVAL_MS - Execute discovery query if under target peers - Execute discovery query if need peers on some subnet: TODO - Disconnect peers if over target peers
