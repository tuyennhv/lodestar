    Class: Eth2Gossipsub¶

    Wrapper around js-libp2p-gossipsub with the following extensions: - Eth2 message id - Emits GossipObject, not InMessage - Provides convenience interface: - publishObject - subscribeTopic - unsubscribeTopic - handleTopic - unhandleTopic

    See https://github.com/ethereum/consensus-specs/blob/v1.1.10/specs/phase0/p2p-interface.md#the-gossip-domain-gossipsub
